| Property | Value |
|----------|-------|

## Spacing Scale

Declared values (multiples of 4). Tailwind CSS 4 default spacing utilities apply. These are the project-specific semantic usages:

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, card internal gaps |
| md | 16px | Default element spacing, mobile horizontal padding |
| lg | 24px | Card padding, responsive horizontal padding (sm breakpoint) |
| xl | 32px | Section heading to content gap, responsive horizontal padding (lg breakpoint) |
| 2xl | 48px | Not used in Phase 1; reserved for future section breaks |
| 3xl | 64px | Section vertical padding (mobile: `py-16` = 64px) |
| 4xl | 80px | Section vertical padding (sm: `py-20` = 80px) |
| 5xl | 96px | Section vertical padding (lg: `py-24` = 96px) |

Exceptions:
- Button minimum height: 48px (accessibility tap target per A11Y-02, uses `min-h-12` in Tailwind)
- Container max-width: 1152px (`max-w-6xl` in Tailwind) -- chosen to keep line lengths readable while filling widescreen viewports

## Typography

### Font Mapping (from design philosophy)

- **Serif (Lora):** Words that carry meaning -- headings, the name, the promise, step numbers
- **Sans-serif (Instrument Sans):** Practical information -- body text, labels, buttons, descriptions

### Scale

| Role | Font | Size | Weight | Line Height | Letter Spacing | Tailwind Classes |
|------|------|------|--------|-------------|----------------|-----------------|
| Body | Instrument Sans | 16px | 400 (regular) | 1.625 (26px) | normal | `font-sans text-base font-normal leading-relaxed` |
| Body small | Instrument Sans | 14px | 400 (regular) | 1.5 (21px) | normal | `font-sans text-sm font-normal leading-normal` |
| Label / Button | Instrument Sans | 16px | 600 (semibold) | 1.25 (20px) | 0.01em | `font-sans text-base font-semibold leading-tight tracking-wide` |
| Section heading (H2) | Lora | 28px | 700 (bold) | 1.2 (33.6px) | -0.01em | `font-serif text-[28px] font-bold leading-tight tracking-tight` |
| Section heading (H2) desktop | Lora | 36px | 700 (bold) | 1.2 (43.2px) | -0.01em | `font-serif text-[28px] lg:text-4xl font-bold leading-tight tracking-tight` |
| Display / H1 | Lora | 36px | 700 (bold) | 1.15 (41.4px) | -0.02em | `font-serif text-4xl font-bold` |
| Display / H1 desktop | Lora | 48px | 700 (bold) | 1.1 (52.8px) | -0.02em | `font-serif text-4xl lg:text-5xl font-bold` |

### Font Weights Loaded

| Font | Weights | Rationale |
|------|---------|-----------|
| Lora | 400-700 (variable) | Variable font, full range auto-included; 400 for italic slogan, 700 for headings |
| Instrument Sans | 400-700 (variable) | Variable font, full range auto-included; 400 for body, 600 for buttons/labels |

### 60/30/10 Distribution

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#FFFFFF` white | Page background, primary sections |
| Secondary (30%) | `#F5F0E8` ecru | Alternating section backgrounds, creating visual rhythm |
| Accent (10%) | `#6B4C3B` brun | Primary text, primary button fill, card borders, interactive focus rings |
| Decorative gold | `#C5973B` or / `#D4A574` or-light | Step numbers (METH-03), decorative dividers, hover highlights -- NEVER for text on light backgrounds |
| Warm accent | `#B8754C` argile | Decorative only -- icon accents, subtle underlines -- NEVER for text on light backgrounds |
| Destructive | `#B91C1C` (Tailwind red-700) | Error states only (Phase 1 has no destructive actions) |

### Contrast Matrix (WCAG AA verified)

| Text Color | Background | Ratio | Status |
|------------|------------|-------|--------|
| brun `#6B4C3B` | white `#FFFFFF` | ~6.8:1 | PASS AA |
| brun `#6B4C3B` | ecru `#F5F0E8` | ~5.2:1 | PASS AA |
| brun `#6B4C3B` | sable `#E8DCC8` | ~4.5:1 | PASS AA (borderline, use cautiously) |
| white `#FFFFFF` | brun `#6B4C3B` | ~6.8:1 | PASS AA (primary button text) |
| or `#C5973B` | white `#FFFFFF` | ~3.2:1 | FAIL -- decorative only |
| or `#C5973B` | ecru `#F5F0E8` | ~2.8:1 | FAIL -- decorative only |
| argile `#B8754C` | white `#FFFFFF` | ~3.5:1 | FAIL -- decorative only |

Accent reserved for: primary button background (brun), focus ring outlines (brun), card subtle borders (brun at reduced opacity or sable), link underlines (brun). Gold/argile are strictly decorative (step numbers, dividers, icon coloring at large sizes).

## Test Page Contract

The Phase 1 test page (`src/app/page.tsx`) must display:

1. **Color swatches:** All 7 palette tokens rendered as visible color blocks with hex labels
2. **Typography specimens:** H1, H2, body, body-small, label -- each with correct font, size, weight
3. **Button pair:** Primary and secondary variants side by side
4. **Card pair:** One card on white background, one on ecru background
5. **Section pair:** One white section, one ecru section -- demonstrating alternating rhythm
6. **Container:** Visible max-width behavior at different viewport sizes
7. **Responsive check:** Layout must not overflow or break at 375px mobile viewport
